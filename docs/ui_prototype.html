Mamba: Linear-Time Sequence Modeling with Selective State Spaces

Gu A, Dao T. arXiv:2312.00752, 2024. 📄 PDF 🌐 arXiv 📊 引用 423

摘要

We introduce Mamba, a new state space model architecture that achieves linear-time sequence modeling through a novel selective scan mechanism. Unlike Transformers with quadratic attention complexity, Mamba maintains computational efficiency while matching or exceeding Transformer performance across multiple benchmarks including language modeling, audio processing, and genomics.
